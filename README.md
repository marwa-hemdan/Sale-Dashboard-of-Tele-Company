## üìä Analysis Summary & Key Metrics

The analysis provides a comprehensive view of the sales process, from initial contact to deal closure.

### Key Performance Indicators (KPIs)

| Metric | Definition |
| :--- | :--- |
| **Total Calls** | The total volume of outbound calls made. |
| **Calls Reached** | The number of calls that resulted in a meaningful contact/interaction. |
| **Deals Closed** | The number of successful sales closed by the agent. |
| **Deal Value ($)** | The total revenue generated by the agent's closed deals. |
| **Call Conversion Rate (%)** | The ratio of Deals Closed to Calls Reached. |
| **Call Drop Rate (%)** | The percentage of calls that were disconnected prematurely. |
